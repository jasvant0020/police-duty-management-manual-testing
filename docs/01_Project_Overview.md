Project Name:
Police Duty Management System 2.0

Project Type:
Web-Based Duty Management Platform

Domain:
Law Enforcement Operations

Objective:
To manage police staff deployment, VVIP security,
real-time location tracking, duty assignment, and
centralized communication through a role-based system.

Technology Stack:

Backend:
- Django
- Django Channels

Frontend:
- HTML
- Tailwind CSS
- JavaScript

Database:
- PostgreSQL / SQLite

Realtime:
- WebSockets
- Redis

Notifications:
- Firebase Cloud Messaging

Roles:

1. Master Admin
2. Super Admin
3. Admin
4. GD Munsi
5. Field Staff
6. VVIP
