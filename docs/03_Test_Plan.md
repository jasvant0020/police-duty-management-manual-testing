# Test Plan

## Project Information

Project Name: Police Duty Management System 2.0

Project Type: Web Application

Domain: Law Enforcement Duty Management

Testing Type: Manual Testing

Prepared By: Jasvant

Version: 1.0

---

## Objective

To verify that the Police Duty Management System functions correctly according to business requirements and provides secure role-based access, duty management, notification delivery, and live tracking functionality.

---

## Scope of Testing

The following modules will be tested:

- Authentication
- User Management
- Role Based Access Control
- Duty Management
- Notification Management
- Live Tracking
- Profile Management

---

## Testing Types

- Functional Testing
- UI Testing
- Validation Testing
- Role-Based Access Testing
- Security Testing
- Regression Testing
